Annotations
 
Enumeration of Energistics quantity classes from the Unit of Measure Dictionary. Each quantity class represents a physical dimension or dimensionless ratio that can be measured. The quantity class defines what is being measured (e.g., length, pressure, volume per time), while units within that class define how it is measured (e.g., metres, feet, inches for length). This enumeration provides a comprehensive list of all standard quantity classes defined in the Energistics Unit Dictionary, including fundamental SI quantities, derived quantities, dimensionless ratios, and specialized petroleum/geotechnical quantities.
