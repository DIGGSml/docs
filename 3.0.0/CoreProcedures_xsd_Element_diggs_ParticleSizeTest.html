Element diggs:ParticleSizeTest
Namespace http://diggsml.org/schemas/3
Annotations
 
An object describing the properties of a laboratory particle size test
procedure. This test produces the distribution of particle sizes in soils as
determined by sieve analysis and/or hydrometer analysis.
