Element diggs:StratigraphySystem
Namespace http://diggsml.org/schemas/3
Annotations
 
An object containing a system of stratrigraphy observations. Observations
within a stratigraphy system must not overlap, and stratUnit values within a system
cannot repeat (eg. they must be unique within the system.
