Element diggs:MonitorType / diggs:monitorLocation
Namespace http://diggsml.org/schemas/3
Annotations
 
Identifies the location associated with the monitoring
measurement.This may be the detector or sensor location, or the
response zone (such as a well screen interval) over which the
measurement applies. In the case where the location is a gemetry
aggregate, the monitor results will refer to all locations as a
whole. In cases where a sensor has detectors in multiple locations,
eqch of which is monitoring properties at the detector location, the
monitor results for each detector should be contained within a
unique, separate Monitor object.
