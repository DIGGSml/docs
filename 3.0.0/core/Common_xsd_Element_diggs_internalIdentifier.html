Element diggs:internalIdentifier
Namespace http://diggsml.org/schemas/3
Annotations
 
The value of this property is the identifier originally created by the
authority for this feature, or that is used as the identifier in the authority's
internal data management system - commonly this might be a borehole number or id,
job number, or contract number. There is no assumption as to the unique scope or
persistence of this identifier. In cases where the original or internal identifier
does not meet the semantic requirements for gml:id, the internal identifier may be
used in order for the originating authority to "map back" the identifier to an
internal data management system. The codeSpace attribute must be populated to
identify the authority that uses the internalIdentifier value.
 
The internal identifier for this feature.
