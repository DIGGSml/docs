Element diggs:GroutBatch
Namespace http://diggsml.org/schemas/3
Annotations
 
A specialization of Sample that contains properties describing the
characteristics of a batch of grout used for production purposes. This object must
reference a DesignGroutMix object defined within the GroutingProgram
feature.
