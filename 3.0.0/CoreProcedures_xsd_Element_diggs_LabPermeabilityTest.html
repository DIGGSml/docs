Element diggs:LabPermeabilityTest
Namespace http://diggsml.org/schemas/3
Annotations
 
An object describing the properties of a laboratory permeability test
procedure. The objective of the permeability test is to determine the coefficient of
permeability of the soil. Several different laboratory testing methods can be
performed to more accurately determine permeability, including constant-head and
falling-head types.
