Element diggs:ColorType / diggs:abundanceCode
Namespace http://diggsml.org/schemas/3
Annotations
 
A measure of the relative amount of the color
present. Terms such as few, rare, common, etc. represent
relative abundance as a range of percentages that are specific
to different standards bodies, so the use of these terms should
reference a codelist dictionary using the codeSpace attribute..
Specific percentage estimates should use the abundancePercent
and/or maxAbundancePercent properties.

range - eg. 30-50%, this property should report the lower

Element diggs:ColorType / diggs:preface
Namespace http://diggsml.org/schemas/3
Annotations
 
A string value that would be used to preface the color
code or name in a programmatically constructed description for
display purposes. Values like "with", "trending towards" or "and"
are commonly used. For example, a secondary color of gray might have
a preface of "with". When combined in a description with a primary
color of brown would produce the phrase "brown with
gray"
