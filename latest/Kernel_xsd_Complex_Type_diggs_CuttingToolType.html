Element diggs:CuttingToolType / diggs:depthInterval
Namespace http://diggsml.org/schemas/2.6
Annotations
 
In cases where the drill pipe is changed out during drilling of the hole, this property contains a LinearExtent object identifying the depth interval where this drill pipe was used. This property is optional and need not be used where the same drill pipe is used for an entire hole.
