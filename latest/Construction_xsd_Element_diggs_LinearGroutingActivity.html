Element diggs:LinearGroutingActivity
Namespace http://diggsml.org/schema-dev
Annotations
 
A feature that contains properties associated with grouting that occurs
within a single linear sampling feature, typically a borehole or sounding. A
grouting activity feature can accommodate all grouting within a sampling feature,
where the grouting setup is the same for all grout stages. Otherwise, multiple
grouting actvities can be assigned to a sampling feature to accommodate variations
in grout setup, equipment, mix types, or where there is significant time between
various grouting activities.
