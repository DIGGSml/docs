A value that identifies this set of Advancement objects to account for redrilling over a specific depth range. Each set of Advancement objects corresponds to a single drilling event over a speciic depth range that encompasses the total depth range of the set of Advancement objects. Possible values for setName could be "Initial", "Redrill 1", etc.
