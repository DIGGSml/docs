Element diggs:AbstractGP_SourceType / diggs:isGroup
Namespace http://diggsml.org/schemas/2.6
Annotations
 
if isGroup is true, this implies that the properties of this source object apply to more than one individual sources. Where true, the noSources attribute must be instantiated with the total number of individual sources that belong to the group. isGroup is used where there are multiple identical soureces that are identical and do not need to be identified individually in configurations or platforms. If false, this indicates that the object consists of only one piece of equipment.
