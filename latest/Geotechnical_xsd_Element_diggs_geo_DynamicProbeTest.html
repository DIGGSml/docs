Element diggs_geo:DynamicProbeTest
Namespace http://diggsml.org/schema-dev/geotechnical
Annotations
 
An object describing a driven penetration test procedure. The Driven Penetration Test includes all methods that involve driving a
rod (with or without a sampler) by impact hammer. The most common test of this type
inclue the Standard Penetration Test and the Becker Hammer Test.
