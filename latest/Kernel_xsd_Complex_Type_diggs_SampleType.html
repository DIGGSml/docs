Element diggs:SampleType / diggs:relativeReferencingSystem
Namespace http://diggsml.org/schemas/2.6
Annotations
 
For a core or block sample, this property can be used to
introduce a relative spatial reference system for describing the
sample relative to a position on the sample instead of relative to
the sampling feature's origin. In this case, the linearElement
property of LinearSpatialReferenceSystem or
VectorLinearSpatialReferenceSystem would reference the sample's
location instead of the sampling feature's.
