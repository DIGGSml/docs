Annotations
 
A value that describes the method by which the observation or measurement
was determined. Allowed values are: - assigned; value is assigned a priori as a
commonly used value or by assumption - calculated; value computed directly from
other value(s) - extrapolated; value estimated from trends in the known data -
graphic; value is manually picked from a graph or visual display - inferred; value
estimated from some indirect measurements such as from a wireline log or transducer
- interpolated; value is estimated betweem two known values using some presumed
function - maximum; value is the largest value inferred for the property - measured;
determined directly from a laboratory or in situ test - minimum; value is the
smallest value inferred for the property - missing; method known but not reported
here - modeled; value estimated from a representation of the measured feature -
moving average; calculated as a moving average of readings - other; method other
than the choices in this list. Further information should be provided in theremarks
property - reported; description comes from a third party source - unknown; method
of determination unknown and can likely not be determined - visual; value comes from
visual inspection
