Element diggs:SourceLocationsType / diggs:sourceRef
Namespace http://diggsml.org/schemas/2.6
Annotations
 
A reference to a GP_Source object's gml:ID. At least one
GP_Source object must exist. If multiple sourceRef properties are
used the sourceLocations property must contain either a
MultiPointLocation or RectifiedGrid object and the order of the
tuples must correspond with the order of the sourceRef properties
within the instance.
