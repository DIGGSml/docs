Attributes
QName Type Use Annotation
character diggs:ColorCharacterEnumType optional 
Describes the character of the color(s) in the associated
unit, from the following enumerated list: banded - multiple colors occur
in horizontal layers within the unit mottled - unit consists of a
pattern of irregular spots, blotches or patches of a different color
than the unit matrix color multicalored - unit consists of multiple
colors where there is no description of the pattern or character of the
various colors uniform - unit consists of a single color variegated - a
multicolored unit containing spots, streaks, or blotches of multiple
colors
gml:id ID required 
Database handle for the object. It is of XML type ID, so is constrained to be
unique in the XML document within which it occurs. An external identifier for the object in
the form of a URI may be constructed using standard XML and XPointer methods. This is done
by concatenating the URI for the document, a fragment separator, and the value of the id
attribute.
origin diggs:ColorOriginEnumType optional 
A term from an enumerated list that identifies the location
(eg matrix or mottle), or the source/cause of the color described.
Defined terms are: matrix - the color of the soil or rock matrix mottle
- the color of soil mottles clay depletion - color caused by depletion
of clay iron depletion - color caused by depletion of iron iron Fe2 mass
- color caused by masses of reduced iron iron Fe3 mass - color caused by
masses of oxidized iron iron-manganese mass - color caused by masses of
iron and manganese jarosite mass - color caused by masses of jarosite
